# Corn Remote Sensing Index Comparison

This repository contains two image comparison sets from early-stage remote sensing research on corn fields. Each set compares the original remote sensing image with the corresponding NDVI and NDWI raster maps.

## Contents

- `Preliminary Result20250826.png`: Comparison of original remote sensing image with NDVI and NDWI raster maps before corn maturity.
- `Preliminary Result20251020.png`: Comparison of original remote sensing image with NDVI and NDWI raster maps after corn maturity.

## Purpose

These images illustrate the changes in vegetation indices (NDVI and NDWI) before and after corn maturity, providing a visual reference for understanding crop growth dynamics. They serve as foundational material for subsequent research on corn maturity prediction using generative AI and remote sensing data.
